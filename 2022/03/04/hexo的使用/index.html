Hexo的使用教程

通过git+github+notejs+hexo搭建好自己的博客后，后续的维护更是需要花费时间，本文将会介绍一些简单常用的hexo命令和运营博客的技巧。

Hexo命令

  • hexo new “文章名” //创建一篇新的文章
  • hexo new page “页面名” //新建页面
  • hexo g //生成页面
  • hexo d //部署
  • hexo g -d //生成页面并部署
  • hexo s //本地预览
  • hexo clean //清除缓存

Hexo常见问题

error：spawn failed

1、删除.deploy_git文件夹

2、输入git config --global core.autocrlf false

3、依次执行

hexo clean

hexo g

hexo d
